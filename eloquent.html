Day 1

Day 1

  • There are 3 special numbers: Infinity, -Infinity, NaN>. For example, 0/0 and Infinity - Infinity will yield NaN
  • Backtick-quoted strings are called template literals. We can embed a computational result by putting it inside ${} in a template literal
  • undefined and null are two empty values in JavaScript that have no different meaning
  • The only ternary operator in JavaScript is x ? y : z. If x is true, then y, else z
  • use === (precisely equal) and !== (not precisely equal) to prevent automatic type conversion
  • When using && and ||, we can make use of short-circuit evaluation (the expression to the right of the operator is only evaluated when necessary). This property allows us to use || to fall back on a default value, i.e. if the value at the left is null/undefined, it will fall back on the value at the right side

Day 2

  • The rules for when a semicolon can be omitted are complex
  • The keywords let, var and const defines a binding. var will not be used as much in the book as it contains some confusing properties. const stands for constant and points to a value that does not change.
